# Navly v1 Open-Source Stack

本目录整理 `Navly_v1` 参考与依赖的开源项目清单，并把 `openclaw/` 的本地源码位置纳入统一入口。

## 当前文件

- `2026-04-06-navly-open-source-dependencies.md`
  - Navly_v1 的 upstream 分层采用策略：第一阶段核心、第一阶段可选扩展、后续增强、基础设施参考
- `openclaw-local-source.md`
  - 仓内 OpenClaw 源码的定位与使用原则
- `repo-archive-status.md`
  - 开源参考项目的本地归档状态与阶段分类

## 使用原则

- 本目录用于说明“依赖什么、为什么依赖、依赖放在架构哪里”
- 本目录不是实现文档，不记录具体代码改动
- 与 Navly_v1 架构文档配合阅读效果最佳
- 归档存在不等于第一阶段必须全部接入运行时
